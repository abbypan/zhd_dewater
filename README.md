zhd_dewater
===================

纵横道( http://www.zonghengdao.net ) 贴子脱水

安装：http://userscripts.org/scripts/show/183860

以 clare [【叶昭】 嘉佑年间凋零故事](http://www.zonghengdao.net/read.php?tid=20984) 为例
